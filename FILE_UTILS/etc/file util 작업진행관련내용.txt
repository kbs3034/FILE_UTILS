1.작업 내용

- 자바 어플리케이션으로 파일 유틸 만들기

2.개발하게된 이유

- 인코딩 변환 유틸의 경우 프로젝트를 진행하며 인코딩이 EUC-KR 프로젝트 파일을 UTF-8로 변환하는 경우가 있어 변환을 위해 유틸을 생성했습니다.

- 파일 확장자 수집 유틸의 경우 프로젝트를 진행하면서 웹서버와 WAS 분리된 환경에서 MIME TYPE 설정이 안되어 정적리소스들에 대해 404 에러가 발생하는 경우가 자주 발견되어
한번에 정리하여 MIME TYPE 설정을 하는게 좋겠다고 생각하여 4개의 어플리케이션이 가진 모든 파일의 확장자를 유틸을 동해 수집하고자 개발하였습니다.
